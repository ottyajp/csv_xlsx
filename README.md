##ぶっちゃけさ

##つかいかた

`git clone`してきて、あとは`./test.sh [InputFile] [OutputFile]`するだけさ。

てきとうにデータをいじくり回した挙句、それっぽいデータが出来上がるはず。拡張子も含めて自分で入れてね。
